# Python Basic Assignment

This repository contains basic Python programs demonstrating:

- Conditional statements
- Dictionary operations
- File handling

## Files Description

- `grade_checker.py`  
  Checks student grade using if-else conditions.

- `student_grades.py`  
  Manages student grades using dictionary operations.

- `write_file.py`  
  Writes data to a text file.

- `read_file.py`  
  Reads and displays data from a text file.

## How to Run

```bash
python grade_checker.py
python student_grades.py
python write_file.py
python read_file.py
```
